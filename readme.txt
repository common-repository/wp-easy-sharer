WP Easy Sharer is a wordpress plugin that adds social media links to at the end of each post and page without having to deal with any code at all.

== Description ==

Description: WP Easy Sharer is a wordpress plugin that adds social media links to at the end of each post and page without having to deal with any code and editing your theme files.

== Installation ==

1. Extract `wp-easy-sharer.zip` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Then you are done.

== Frequently Asked Questions ==

*What does this plugin do?
It directly adds facebook, twitter, digg, delicious, reddit 
and stumbleupon sharing buttons to at the end of each post and page.
